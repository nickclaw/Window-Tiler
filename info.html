Frequently Asked Questions

Why isn't it working on multiple screens for me?

The displays API is relatively new to Chrome and has a wide range of support across operating systems. But don't worry, the Chrome team is continuing to improve it and support should be coming soon. Chrome versions 33 and above seem to work completely.

How can I control where each window goes?

While there is no way to control exactly where each browser window will end up. You can easily make suggestions. Simply drag a browser window to roughly where you would like to go and the algorithm will try to make sure it stays there.

Why aren't there more layouts?

First of all know that there is often more than 4 layouts per number of windows, try scrolling down to see more options. If there is a layout that you feel is missing feel free to contact me.

How can I help?

If you have good ideas for this extension or notice a bug that just keeps popping up feel free to shoot me an email or leave a comment on the Chrome store! If you're more technically minded, create an issue or pull request on GitHub.
